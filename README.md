# CodeAlpha_FlashcardQuizApp

An interactive flashcard study tool to help you memorize anything effectively.

## ✨ Features
- Flip cards to reveal answers (click the card or "Show Answer" button)
- Navigate between cards using Next / Previous buttons
- Add new flashcards with custom questions and answers
- Edit or delete existing flashcards
- Progress bar showing current position in the deck
- Clean and user-friendly interface

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 How to Run
1. Download `index.html`
2. Open it in any web browser — no installation needed!
